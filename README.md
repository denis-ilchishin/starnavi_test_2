# starnavi_test_2

### Guide
- Clone repository
```
git clone git@github.com:denis-ilchishin/starnavi_test_2.git
cd starnavi_test_2
```
- Install
```
make install
```
- Run tests
```
make test
```
- Start up dev server on http://localhost:8000/
```
make start
```
